Follow the Visual Identity strictly:
- **RED GOAL**: `#D72E1F`
- **ASPHALT BLACK**: `#191919`

## Design Direction

The visual style must be:
- Cinematic
- Motorsport-oriented
- Technical
- Premium
- Dark
- High contrast
- Modern
- Urban
- Professional
- Conversion-focused

Use **ASPHALT BLACK** as the main background, **RED GOAL** for accents and CTAs, and **WHITE SMOKE** for readable text sections.

Use the gradient from **ASPHALT BLACK** to **RED GOAL** for hero overlays, buttons, section dividers, and premium visual effects.

Avoid generic corporate design. The page should feel like a professional motorsport academy, not a standard education website.

## Technical Requirements

- Make the page responsive.
- Prioritize mobile-first layout.
- Ensure strong accessibility and readable contrast.
- Use reusable components where appropriate.
- Keep the UI clean, structured, and easy to maintain.
- Use semantic HTML.
- Add hover states and smooth transitions.
- Use placeholder images if no assets are available.
- Keep forms visually prominent and easy to complete.
